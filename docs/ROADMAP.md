# OpenDataFitHou Development Roadmap

## Current Version: 1.0.0

### Completed ✅
- Thu thập dữ liệu từ OpenStreetMap qua Overpass API
- Chuyển đổi GeoJSON sang RDF/Turtle format
- Tích hợp với Wikidata để bổ sung thông tin
- Xây dựng ontology cho dữ liệu địa lý Hà Nội
- Batch processing cho nhiều loại amenity
- Docker support
- Documentation cơ bản
